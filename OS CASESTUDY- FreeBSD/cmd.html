COMMANDS-FREE BSD

vmstat -i
Displays statistics about system interrupts
Interrupt number, count, device name, rate

df -h | sort -k 5 -r
Shows disk usage of all mounted file systems and sort by usage percentage

kldstat | sort
Lists kernel modules and sort them alphabetically

w
Lists all users currently logged in and their sessions

kldstat -v | head -n 5
Displays loaded kernel modules with detailed info. and only shows the first 5 lines

kldstat | wc -c
Lists the kernel modules and show the total number of characters in the output

dmesg | tail -n 10
Display the last 10 lines of the system message buffer

ifconfig -a
Used to display detailed information about all network interfaces on the system

iostat -x
Display detailed information about the system’s disk I/O activity.

netstat -rn
Display detailed information about networking routing tables

swapinfo
Detailed information about memory usage and swap space

cat /var/log/messages | grep -i error
Display the contents of a log file and filter for error messages

uptime
Display system uptime and load averages

find /home -type f -mtime -7 | less

uname -a
Detailed information about the system’s CPU architecture

pstat -T
Display a summary of system resource usage by process

df -h
Check available space on mounted file systems

last reboot
Show the last few system reboots

ping <hostname>
Test network connectivity and diagnose network issues using ping
